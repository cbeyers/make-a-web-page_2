Lecture 1

1. HTML are the major file types/documents on the web. These are:
  • Text
  • Markup
  • links to images/video/other pages
2. The standard format of an element is: <name> contents</name>
3. The element <a> is used to add a link to a word. An example is: Take a break!
4. Elements can be looked up online and need not be memorised.
5. Some elements have no text/context and do not need to be closed. e.g. <img>
6. While there are several elemnts, the major thing to remeber is that they can be either inline or block.
7. Block element have a 'size' where inline elements are a continuious line of text.

Lecture 3

1. CSS stand for Cascading Style Sheet
2. Styles are inherited in all offspring elements.
3. Boxes of HTML code should be defined by <div>(block) or <span>(inline) tags to allow easy CSS coding to reduce repetition.
4. CSS uses sectors to which the style rule is applied. Alternatively a class selector can be used.
CSS format: Attribute:value; e.g. color:red;
5. CSS helps to avoid repetition as defined elements do not need to be formatted individually, but can be grouped in the same class to facilitate changes to style. Avoiding repetition saves time through reducing the amount of code needed for formatting. Errors arise if we repeat the same code and can mean that changing code will be clumsy.
6. All code such as HTML should be structured through indentation/levels to make it easier to see grouped elements and determine their use or function.
